String Substitution - Strings (%s)


                        const name = "Charles";
                        console.log("Hi! My name is %s.", name);
There a four placeholders that can be used in console output. %s is used to substitute a string. %d is used to substitute a number. %o is used to substitute and object. %c ymmv, is used to substitute css. Anyone have a good use fir this?

String Substitution - Numbers (%d)


                        const n = 42;
                        console.log("The answer to the Ultimate Question is %d.", n);

String Substitution - Objects (%o)


                        const person = { Name: "Chuck D" };
                        console.log("Hi! I am %o", person);

String Substitution - CSS (%c)


                        console.log("Anyone have a good use for %cthis one?", "background-color: blue");

Organizing output


                        console.log("This is the outer level");
                        console.group("Group");
                        console.log("Level 2");
                        console.groupCollapsed("Collapsed");
                        console.log("Level 3");
                        console.warn("More of level 3");
                        console.groupEnd();
                        console.log("Back to level 2");
                        console.groupEnd();
                        console.log("Back to the outer level");
Grouping things simply indents output until end group.

Viewing Data Objects


                    const people = [
                        { "Name": "Charles B", "Title": "Professional", "Skills": ["All the things"] },
                        { "Name": "Chuck D", "Title": "Public Enemy", "Skills": ["Musician"] }
                    ];

                    console.table(people);
Tables can sort on columns.

Viewing Data Objects - Specify Columns


                    const people = [
                        { "Name": "Charles B", "Title": "Professional", "Skills": ["All the things"] },
                        { "Name": "Chuck D", "Title": "Public Enemy", "Skills": ["Musician"] }
                    ];

                    console.table(people, ["Name"]);
